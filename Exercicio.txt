Criar um repositório no Github
$ abrir o site https://github.com/
$ logar com usuario e senha do github
$ Clicar no sinal de "+ (mais) no canto superior direito" / clicar em "New repository"
$ no combo Repository name, digitar o nome do repositório que deseja Criar
$ no campo "Description (optional)" digitar no combo uma breve descrição do que se trata o repositório
$ selecionar se o repositorio é "Publico"  ou   "Privador"
$ selecionar a caixa "Add a README file"
$ selecionar a opcao "add .gitignore" OBS: Selecione a linguagem a qual voce estara desenvolvendo ou crie seu arquivo 
$ Click em "Create Repository" para criar repositorio "
Conectar os dois repositórios
